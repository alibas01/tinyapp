## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parse
- cookie-session
- morgan
- nodemon
- method-override

## Getting Started

- Install all dependencies (using the `npm install <package> --save` command).
- For dev dependencies please use (`npm install <package> --save-dev`) command.
- Run the development web server using the `npm start` command.
